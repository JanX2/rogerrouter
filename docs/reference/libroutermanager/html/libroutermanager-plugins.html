#define             ROUTERMANAGER_PLUGIN_REGISTER(TYPE_NAME, TypeName, type_name)

Registers a plugin with the RouterManager plugin system, including registering the type specified in the parameters and declaring its activate and deactivate functions.

Registers a configurable plugin with the RouterManager plugin system, including registering the type specified in the parameters and declaring its activate and deactivate and widget creation functions.
